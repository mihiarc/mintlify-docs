pyFIA is a Python library for analyzing USDA Forest Inventory and Analysis (FIA) data. Built on DuckDB and Polars, it delivers design-based population estimates — trees per acre, volume, biomass, carbon, mortality, and more — that match EVALIDator output exactly, at 10–100× the speed of the web interface.

## Get started in 3 steps

<Steps>
  <Step title="Install pyFIA">
    ```bash
    pip install pyfia
    ```
  </Step>
  <Step title="Download state data">
    ```python
    from pyfia import download
    db_path = download("RI")  # Rhode Island — small and fast
    ```
  </Step>
  <Step title="Run your first estimate">
    ```python
    from pyfia import FIA, area, volume

    with FIA(db_path) as db:
        db.clip_most_recent()
        print(area(db, land_type="forest"))
        print(volume(db, grp_by="SPCD"))
    ```
  </Step>
</Steps>

<Note>
  Always call `clip_most_recent()` or `clip_by_evalid()` before running estimates. Without an EVALID filter, plots are counted across multiple overlapping evaluations and results will be incorrect.
</Note>
